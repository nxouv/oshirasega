# おしらせ画

SNS投稿用の白背景の文章画像をかんたんに作成できるWebアプリです。

## 🔗 URL

https://oshirasega.nanataro.app

## ✨ 特徴

- **4種類のフォント** - 明朝体、ゴシック、丸ゴシック、手書き風から選択
- **5種類の背景色** - 白、グレー、ピンク、ブルー、ダークに対応
- **AIによる文章チェック** - 誤字チェックと表現の改善機能
- **自動ページ分割** - 長文は自動で複数ページに分割
- **スマホ対応** - PCでもスマホでも快適に利用可能
- **会員登録不要** - すぐに使える

## 🛠 技術スタック

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS
- Google Gemini API

## 📝 使い方

1. テーマを選ぶ（フォーマル、シンプル、やさしい、かわいい）
2. 本文を入力
3. 必要に応じてタイトル、日付、名前を追加
4. 「保存する」で画像をダウンロード
